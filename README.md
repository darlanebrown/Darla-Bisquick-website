A simple multi-page website showcasing biscuit recipes, ordering options, and interactive features using HTML, CSS, and JavaScript.
darlas-biscuit-homepage.png   # Homepage design/mockup image
index.html                    # Main homepage
order-form.html               # Order submission page
recipe.html                   # Biscuit recipe page
myscript.js                   # JavaScript for interactivity
styles.css                    # Styling for all pages
Features

🏠 Homepage (index.html)
Welcoming layout introducing Darla’s biscuits
Navigation to recipe and ordering pages
📖 Recipe Page (recipe.html)
Displays biscuit ingredients and instructions
Clean, readable layout for users
🛒 Order Form (order-form.html)
Allows users to submit biscuit orders
Form inputs for customer details and selections
🎨 Styling (styles.css)
Consistent design across all pages
Responsive and user-friendly layout
⚙️ Interactivity (myscript.js)
Handles form behavior and user interactions
Enhances user experience on the site
How to Run the Project
Download or clone the project files
Open index.html in your browser
Navigate through the site using links
Enhances user experience on the site
